# SmartCart Customer Clustering
## Overview
An AI/ML project using unsupervised learning to segment 2,240 e-commerce customers into distinct groups based on demographics and buying habits.

## Key Features
Demographics: Age, Income, Education, Household size.

Spending: Analysis of Wine, Fruit, Meat, Fish, Sweets, and Gold purchases.

Engagement: Web visits, catalog purchases, and campaign responses.

## Tech Stack
Python (Pandas, Scikit-learn, Seaborn)

Jupyter Notebook

## Workflow
Cleaning: Handled missing income data and outliers.

## Feature Engineering: Created 'Age', 'Total Spending', and 'Parental Status' features.

## Clustering: Applied K-Means/Agglomerative clustering to identify behavior patterns.
